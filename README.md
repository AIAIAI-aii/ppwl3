# ppwl3

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

Temukan 2 perbedaan utama dari Tailwind v3 & v4 yang dapat ditemukan di play.tailwindcss.com? 
(1) v3 ada config, v4 tidak ada
(2) cara kerja konfigurasi dan customizing berbeda karena v4 menggunakan aturan seperti @theme dan variabel CSS, sedangkan v3 biasanya mengatur tema dan konfigurasi kustom lewat file.
